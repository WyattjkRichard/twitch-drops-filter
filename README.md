# Twitch Drops Filter

Filter unwanted games from the Twitch Drops campaigns page with a lightweight Chrome extension.

## What it does

- Hides Twitch Drops campaigns.
- Lets you add and remove blocked titles from the extension popup.
- Adds a small Hide button to each campaign header so you can block titles directly from Twitch.

## Install

This extension is not packaged for the Chrome Web Store, so you need to load it as an unpacked extension.

1. Open Chrome and go to `chrome://extensions`.
2. Turn on Developer mode.
3. Click Load unpacked.
4. Select this project folder.
